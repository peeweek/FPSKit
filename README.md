# FPSKit
First Person Game Kit for Unity 2020.3+

## Requirements

* Unity 2020.3 or newer
* HDRP or URP
* Cinemachine
* New Input System

## Features

* Configurable First Person Controller
  * ✅ Move / Look
  * ✅ Head Bobbing
  * ✅ Crouch
  * ✅ Jump (+ Multiple)
  * ✅ Sprint (Dash)
  * ✅ Aim
  * ✅ Slide on Slopes
* Configurable Input
  * Legacy System
  * ✅ New Input System
  * ✅ Pad Rumble on Shoot
* Camera Rigs
  * ✅ Cinemachine Virtual Cameras
  * Standard Camera
* Attachments (Weapons, Hands, etc.)
  * Simple Locomotion
  * Simple Weapon
  * ✅ Aim Down Sight Weapon
  * Simple Weapon + Alt shot 
* Projectiles
  * Instant Projectile
  * ✅ Rigid Body Projectile
  * ✅ Projectile Audio/Light/VFX
* Pickups
  * Weapons
  * Ammo
