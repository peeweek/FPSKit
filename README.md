# FPSKit
First Person Game Kit for Unity 2020.3+

## Requirements

* Unity 2020.3 or newer
* HDRP or URP
* Cinemachine
* New Input System

## Features

* Configurable First Person Controller
  * Move / Look
  * Crouch
  * Jump (+ Multiple)
  * Sprint
* Configurable Input
  * Legacy/New Input System
  * Pad Rumble on Shoot
* Camera Rigs
  * Cinemachine
  * Standard Camera
* Attachments (Weapons, Hands, etc.)
  * Simple Locomotion
  * Simple Weapon
  * Aim Down Sight Weapon
  * Simple Weapon + Alt shot 
* Projectiles
  * Instant Projectile
  * Rigid Body Projectile
  * Projectile Audio/Light/VFX
* Pickups
  * Weapons
  * Ammo
